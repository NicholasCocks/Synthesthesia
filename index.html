About the project

The Fast Fourier Transform (FFT) which you can see at the bottom left of the screen is an algorithm that converts a signal from it's 'original domain' to a representation in the frequency domain. This algorithm is used in everything from image and audio compression to the frequency analyzer you see when you sketch. This implementation allows you to adjust the number of discrete data points and therefore the width of the bars.
